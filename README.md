
# Stone Canyon Site


## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Channel-Automation-Edge/starfish-pages
   cd your-repository
   ```

2. **Install dependencies**:
   Make sure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

## Running the App

1. **Start the development server**:
   ```bash
   npm run dev
   ```

2. **Open your browser**:
   Open your browser and navigate to `http://localhost:3000` to see the app running.

## Building for Production

1. **Build the app**:
   ```bash
   npm run build
   ```

2. **Serve the built app**:
   You can use a static server to serve the built app. For example, using `serve`:
   ```bash
   npm install -g serve
   serve -s build
   ```
